# Sales Performance Analysis

## Tools Used

* SQL
* Excel

## Project Overview

This project analyzes sales performance data to understand revenue trends, product performance, and customer behavior.

## Key Insights

* Total revenue was driven mainly by a small group of high-value products
* Malaysia contributed the highest share of revenue among countries
* Sales show clear seasonal fluctuations across months
* Top customers account for a significant portion of total revenue

## Dashboard Features

* Revenue trend over time
* Top-performing products
* Revenue by country
* Customer ranking analysis

## Files Included

* Cleaned datasets (customers, orders, products)
* SQL queries used for analysis
* Excel dashboard export (PDF)
